Pytania i odpowiedzi dotyczące budowy selektora

  1. 1. Wymienić oraz opisz dwie części selektora.

    Selektor składa się z dwóch części:

    • Selektor: Określa elementy HTML, do których ma zostać przypisane formatowanie. Może to być znacznik (np. p, h1) lub klasa (np. .klasa).
    • Deklaracja: Zawiera instrukcje formatowania umieszczone w nawiasach klamrowych {...}. Składa się z cech (właściwości) i wartości.

  2. 2. Znaczenie nawiasów klamrowych {...}.

    Nawiasy klamrowe {...} otaczają deklaracje stylów CSS. Wewnątrz nich określa się właściwości (cechy) i ich wartości, które definiują formatowanie wybranego selektora.

  3. 3. Opisz cechy.

    Cechy (właściwości): Są to atrybuty formatowania, które można przypisać do elementów HTML. Przykładami cech są:

    • color – określa kolor tekstu.
    • font-size – określa wielkość czcionki.
    • background-color – określa kolor tła.

  4. 4. Opisz wartości.

    Wartości: Są to konkretne ustawienia dla danej cechy. Mogą to być:

    • Kolory, np. red, #FFFFFF.
    • Jednostki miar, np. 20px, 10pt.
    • Procenty, np. 50%.

  5. 5. Kiedy stosujemy znak średnika (;), kiedy nie jest konieczny?

    Znak średnika (;) oddziela poszczególne deklaracje w ramach bloku stylu. Jest obowiązkowy między kolejnymi deklaracjami. Nie jest wymagany po ostatniej deklaracji w bloku.

    Przykład:

    p {
    color: red; /* Średnik jest wymagany */
    font-size: 20px; /* Średnik jest wymagany */
    background-color: yellow /* Średnik nie jest konieczny */
}
