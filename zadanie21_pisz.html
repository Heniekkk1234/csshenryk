Zadanie 21: Użycie ID

Pojęcie Znaczenie (uzupełnia uczeń)
Co znaczy skrót ID? Unikalny identyfikator przypisany do elementu HTML
Jaki jest cel stosowania ID? Umożliwia zróżnicowanie stylów dla konkretnych, pojedynczych elementów
Jakie jest znaczenie znaku #? Symbol # w CSS oznacza identyfikator (ID)
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? ID nadaje się w HTML poprzez atrybut: id="nazwa"
W CSS styl definiujemy przez: #nazwa { ... }
Styl zostanie zastosowany tylko do elementu o tym ID
Uwagi na temat stosowania ID
  1. ID może wystąpić tylko raz w całym dokumencie
  2. Elementy z tym samym ID to błąd — ale strona może się mimo to wyświetlić
  3. Jeśli dany element może się powtórzyć, lepiej użyć klasy (`class`)
