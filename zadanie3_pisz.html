Cechy modyfikowania formatowania w HTML i CSS

HTML CSS
Modyfikacja wyglądu jest żmudna. Rozdzielenie struktury i prezentacji dokumentów.

Nowe możliwości CSS:

  1. Wycofywanie przestarzałych atrybutów HTML.
  2. Dokumenty są bardziej przejrzyste i krótsze.
  3. Zarządzanie całymi seriami dokumentów.

Wady CSS:

  • Złożona specyfikacja.
  • Brak sygnalizacji błędów.
  • Błędne obsługa przez przeglądarki.
