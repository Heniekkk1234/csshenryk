Pytania i odpowiedzi dotyczące zewnętrznego arkusza stylów

  1. 1. Składnia teoretyczna → otoczony ramką (może być tabela jednokomórkowa, konieczne wcięcia).

    Powyżej przedstawiono składnię teoretyczną podłączenia zewnętrznego arkusza stylów CSS.

  2. 2. Gdzie wstawiany jest styl zewnętrzny, jaki znacznik jest za to odpowiedzialny oraz ile razy może on występować w dokumencie jako podłączenie stylu CSS?

    Styl zewnętrzny jest wstawiany w sekcji dokumentu HTML za pomocą znacznika . Może występować wiele razy w dokumencie, aby dołączyć wiele arkuszy styli.

  3. 3. Jak jest najprostsza forma wstawiania stylu zewnętrznego → otoczony ramką (może być tabela jednokomórkowa)?

    Powyżej przedstawiono najprostszą formę wstawiania zewnętrznego arkusza stylów CSS.

  4. 4. Ile arkuszy styli może być dołączanych do jednego dokumentu HTML oraz jaka zasada musi być zachowana przy większej ilości dołączanych styli?

    Do jednego dokumentu HTML można dołączyć dowolną liczbę arkuszy styli. Zasada, która musi być zachowana, to kolejność ich wczytywania – arkusze dołączone później mogą nadpisywać style z wcześniej dołączonych arkuszy w przypadku konfliktów.

  5. 5. Co się dzieje w przypadku konfliktów? Przez konflikt rozumiemy, gdy te same style są definiowane w różnych arkuszach styli podłączonych do tego samego dokumentu.

    W przypadku konfliktów, style z arkusza dołączonego później mają pierwszeństwo nad stylami z wcześniej dołączonych arkuszy. Jeśli style mają taką samą specyficzność, ostatnio zdefiniowany styl będzie miał priorytet.

  6. 6. Jak są tworzone i edytowane pliki arkuszy z użyciem edytora tekstowego?

    Pliki arkuszy CSS można tworzyć i edytować za pomocą dowolnego edytora tekstowego, np. Notepad, Notepad++, Sublime Text. Wystarczy utworzyć nowy plik, wpisać definicje stylów i zapisać go z rozszerzeniem .css.

  7. 7. Jak są tworzone i edytowane pliki arkuszy z użyciem specjalnego edytora CSS?

    Pliki arkuszy CSS można tworzyć i edytować za pomocą specjalnych edytorów CSS, np. Visual Studio Code, TopStyle Lite, Balthisar Cascade. Edytory te oferują funkcje automatycznego uzupełniania kodu, podświetlanie składni i inne narzędzia ułatwiające pracę.

  8. 8. Jakie rozszerzenie posiadają pliki będące zewnętrznym arkuszem styli?

    Pliki będące zewnętrznym arkuszem styli mają rozszerzenie .css.

  9. 9. Jakie są zalety zewnętrznego arkusza styli (dwie – z użyciem wyliczenia)?

    1. Łatwiejsze zarządzanie wyglądem wielu stron – zmiana w jednym arkuszu CSS wpływa na wszystkie strony, które go używają.
    2. Oddzielenie struktury (HTML) od prezentacji (CSS), co sprawia, że kod jest bardziej czytelny i łatwiejszy w utrzymaniu.

  10. 10. Uwagi na temat pisania w CSS.

    1. CSS dopuszcza stosowanie zarówno dużych, jak i małych liter.
    2. Białe znaki (spacje, tabulacje, znaki nowej linii) nie mają wpływu na działanie arkusza.
    3. Komentarze w CSS pisze się za pomocą znaków /* */.
    4. Należy unikać błędów składniowych, ponieważ nie są one sygnalizowane podczas wykonywania styli.
